# Plan 01-02: Phone Client Connection and Room Joining

## Objective
Build the Phone browser client that reads the Room ID from the URL, joins the correct Socket.io room, and shows a connection status. Update the server to handle `phone:join` and notify the Totem when a phone enters.

## must_haves
- Navigating to `http://[LAN-IP]:3000/phone?room=XXXXXX` shows a connection status page.
- The phone emits `phone:join` with the room ID immediately on load.
- The server joins the socket to that room with `socket.join(roomId)`.
- The Totem's status text changes to "Player connected! ✓" (green) when the phone joins.
- The Totem's status reverts to "Waiting for player..." if the phone disconnects.

## Tasks

### Task 1: Add phone:join handler to server.js

<read_first>
- `server.js` (to see existing Socket.io event handlers and room state)
</read_first>

<action>
Open `server.js` and add the following Socket.io event handlers inside the `io.on('connection', (socket) => { ... })` block, AFTER the existing `totem:ready` handler:

```javascript
  // Phone joins a room
  socket.on('phone:join', ({ roomId: requestedRoomId }) => {
    if (requestedRoomId !== roomId) {
      // Invalid room — session may have expired (Phase 3 will handle this properly)
      socket.emit('server:error', { message: 'Room not found or expired.' });
      return;
    }
    socket.join(roomId);
    socket.data.role = 'phone';
    console.log(`Phone joined room: ${roomId}`);

    // Notify totem
    io.to(roomId).except(socket.id).emit('totem:phone-connected', { socketId: socket.id });

    // Confirm to phone
    socket.emit('server:joined', { roomId });
  });

  // Handle disconnect
  socket.on('disconnect', () => {
    if (socket.data.role === 'phone') {
      io.to(roomId).emit('phone:disconnected', { socketId: socket.id });
      console.log(`Phone disconnected from room: ${roomId}`);
    }
  });
```

**Replace** the existing generic `socket.on('disconnect'...)` handler with the one above (it now handles role-aware disconnect).
</action>

const socket = io();

const statusEl = document.getElementById('status');
const roomDisplayEl = document.getElementById('room-display');
const pingBtn = document.getElementById('ping-btn');
const pongFeedback = document.getElementById('pong-feedback');

// 1. Read room ID from URL query parameter
const params = new URLSearchParams(window.location.search);
const roomId = params.get('room');

if (!roomId) {
  statusEl.textContent = 'Error: No room ID in URL.';
  statusEl.classList.add('error');
} else {
  roomDisplayEl.textContent = `Room: ${roomId}`;

  // 2. Join room when connected to server
  socket.on('connect', () => {
    statusEl.textContent = 'Connecting to room...';
    socket.emit('phone:join', { roomId });
  });
}

// 3. Confirmed in room
socket.on('server:joined', ({ roomId: confirmedRoom }) => {
  statusEl.textContent = `Connected to room ${confirmedRoom} ✓`;
  statusEl.classList.add('connected');
  // Phase 2 will show the ping button here
  // pingBtn.style.display = 'block';
});

// 4. Error from server (e.g. room not found)
socket.on('server:error', ({ message }) => {
  statusEl.textContent = `Error: ${message}`;
  statusEl.classList.remove('connected');
  statusEl.classList.add('error');
});

// 5. Socket disconnected
socket.on('disconnect', () => {
  statusEl.textContent = 'Disconnected...';
  statusEl.classList.remove('connected');
});

// 6. Phase 2 placeholder — ping button handler (wired up in Plan 02-01)
